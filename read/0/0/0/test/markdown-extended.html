Admonition

Admonition 支持。包名：marked-admonition-extension

marked-admonition-extension 支持以下类型:

abstract、attention、bug、caution、danger、error、example、failure、hint、info、note、question、quote、success、tip、warning

格式：

!!! note
Contents
!!!
